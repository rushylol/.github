# Rushy Development

Rushy Development is a small, learning-focused development organization working on projects in **Lua, Luau, Python, and Node.js**.  

The purpose of this organization is to practice programming, explore different technologies, and improve development skills through hands-on projects published on GitHub.

---

## About

Rushy Development is not a commercial company. It is a collaborative learning space where members experiment with code and build small projects while learning modern development workflows.

The organization focuses on:

- Game scripting with **Lua / Luau** (including Roblox development)
- Backend scripting and automation using **Python**
- Web and server development using **Node.js**
- General software engineering fundamentals

---

## Technologies Used

- Lua / Luau  
- Python  
- Node.js (JavaScript / TypeScript)  
- Git & GitHub  

Additional tools and libraries may be introduced over time as learning progresses.

---

## Goals

- Learn real-world development workflows
- Build and publish experimental projects
- Collaborate on small-scale software development
- Track progress over time through GitHub repositories
